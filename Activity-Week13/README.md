Coach Keanu Training Hub

A responsive multi-page basketball training website built using **Bootstrap 5 and JavaScript**.  
This project demonstrates modern front-end development using responsive design, UI components, and form validation.

---

Project Overview

The Coach Keanu Training Hub is a simple basketball training website that allows users to explore training services, view coaching features, and submit booking requests through a contact form with validation.

It is designed using Bootstrap 5 to ensure responsiveness across all devices.

---

Objectives

- Build a responsive multi-page website
- Apply Bootstrap grid system and components
- Implement a working navigation system
- Use JavaScript for form validation
- Create a clean and user-friendly UI

---

Website Pages

Home Page (`index.html`)
- Hero section introducing the training hub
- Overview of coaching services
- Training cards (Skills, Conditioning, Game IQ)
- Navigation menu and footer

---

Features Page (`features.html`)
- Displays basketball training features:
  - Personal Training
  - Skill Development
  - Performance Growth
- Card-based layout with images
- Responsive grid system using Bootstrap

---

Contact Page (`contact.html`)
- Displays coach contact information:
  - Name
  - Email
  - Phone
  - Location
- Includes a **Bootstrap modal form**
- Users can submit training requests
- Form includes validation

---

Technologies Used

- HTML5

---

JavaScript Functionality

The project includes form validation using JavaScript:

- Uses `checkValidity()` to validate inputs
- Prevents form submission if invalid
- Adds `.was-validated` class for Bootstrap styling
- Supports multiple forms using `querySelectorAll`
